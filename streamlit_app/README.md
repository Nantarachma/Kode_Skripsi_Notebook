A Streamlit-based dashboard for simulating and visualizing real-time Network Intrusion Detection using optimized XGBoost models.

## Prerequisites

The following data files are **generated by the training notebook** (`training_nids_xgboost_multiclass.ipynb`) and must be placed in the `streamlit_app/` directory before running:

| File | Description |
|---|---|
| `models_catalog.csv` | Catalog of trained Pareto-optimal models with metadata (method, filename, label, f1_score, latency_us, n_estimators, max_depth). |
| `simulation_data.csv` | Pre-sampled network traffic data used for real-time simulation. Contains feature columns and a `Label_True` column. |
| `label_encoder.pkl` | Fitted `LabelEncoder` for the target variable. Used to derive the label map dynamically. |
| `scaler.pkl` | Fitted `StandardScaler` for feature columns. Used to normalize simulation data before inference. |
| `models/` | Directory containing XGBoost model files (`.json` format, saved via `xgb.save_model()`). Filenames must match those listed in `models_catalog.csv`. |

## Installation

```bash
cd streamlit_app
pip install -r requirements.txt
```

## How to Run

```bash
cd streamlit_app
streamlit run app.py
```

The dashboard will open in your browser (default: `http://localhost:8501`).

## Test Scenarios
